Öğrenci No:250541078
AD-SOYAD:Ecenur Üner
Konu:Basit kart oyunu stratejisi
Lütfen seçtiğiniz algoritmaya ait çözümü ve diğer isterleri aşağıya ekleyiniz:
Başla

// 1. Girdi Al
OyuncuKartları = [3, 5, 6, 9] // örnek kartlar
RakipKartları = [4, 7, 2, 10]
KazanılanTurlar = 0
TurSayısı = Uzunluk(RakipKartları)

// 2. Döngü: Her tur için
Tur = 1
Döngü başlat: Tur <= TurSayısı iken

    RakipKart = RakipKartları[Tur - 1]

    // 3. Koşullu Durum: Rakip kartını yenebilecek en küçük kartı bul
    EnUygunKart = -1
    EnKüçükFark = Sonsuz

    Her kart için OyuncuKart in OyuncuKartları
        Eğer OyuncuKart > RakipKart ise
            Fark = OyuncuKart - RakipKart
            Eğer Fark < EnKüçükFark ise
                EnKüçükFark = Fark
                EnUygunKart = OyuncuKart

    // 4. Karar Ver
    Eğer EnUygunKart != -1 ise
        OyuncuKartları'dan EnUygunKart'ı çıkar
        KazanılanTurlar += 1
        Yazdır("Tur " + Tur + ": Kazanıldı! Oynanan kart: " + EnUygunKart)
    Aksi halde
        EnDüşükKart = Minimum(OyuncuKartları)
        OyuncuKartları'dan EnDüşükKart'ı çıkar
        Yazdır("Tur " + Tur + ": Kaybedildi. Oynanan kart: " + EnDüşükKart)

    Tur += 1

Döngü bitir

// 5. Sonuç
Yazdır("Toplam kazanılan tur: " + KazanılanTurlar)

Bitir
